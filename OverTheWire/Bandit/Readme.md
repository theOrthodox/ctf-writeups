The Bandit from OverTheWire is a beginner-friendly cybersecurity wargame designed to teach the fundamentals of Linux, command-line usage, and basic security concepts.

Bandit is intended for people who are new to Linux and Capture The Flag (CTF) challenges. Each level presents a small task that requires players to use Linux commands, explore the filesystem, and think like an attacker to retrieve the password for the next level.

This repository contains my step-by-step solutions and explanations for the Bandit levels I completed. The goal of these write-ups is to document my learning process and build a strong foundation in Linux enumeration, file handling, and problem-solving techniques commonly used in cybersecurity and penetration testing.
