# 📌 About Natas

**Natas** is a web security wargame focused on:
- Web vulnerabilities
- Input validation flaws
- Authentication bypass
- Command injection
- Encoding/decoding tricks

Each level provides:
- A username & password
- A vulnerable web application
- A goal: retrieve the password for the next level
